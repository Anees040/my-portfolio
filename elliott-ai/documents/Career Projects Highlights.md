## Other/Personal Projects

1. **AlphaListNav.js:**  
- alphaListNav.js is an open source vanilla JavaScript plugin that will add an alphabetical navigation bar to your long HTML lists. Click a letter in the nav to quickly filter the list to show items that match that letter. With many different options and features.  
  AlphaListNav has very similar functionality to jQuery ListNav but has been re-written without the jQuery dependency. It can serve as a drop-in replacement, removing the need for jQuery.
